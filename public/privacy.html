🛡️ MatBil Gizlilik Politikası

Sürüm 1.0 • Yürürlük: 2026-04-24 • 6698 sayılı Kişisel Verilerin Korunması Kanunu (KVKK) uyarınca

Özet: MatBil, matematiksel biliş ve diskalkuli eğitimi için geliştirilmiş bir uygulamadır. Topladığımız veriler yalnızca cihazınızda saklanır. Sunucumuza (premium kod doğrulaması hariç) hiçbir kişisel veri gönderilmez. Üçüncü taraflarla paylaşım yapılmaz.

1. Veri Sorumlusu

MatBil Matematiksel Biliş Uygulaması akademik/eğitim amaçlı olarak Prof. Dr. Yılmaz Mutlu ve Prof. Dr. Sinan Olkun tarafından geliştirilmiştir.

İletişim: destek@matbil.app

2. Hangi Verileri Topluyoruz?

  • Öğrenci profil bilgisi: Ad (veya takma ad), yaş. Oyun deneyiminin kişiselleştirilmesi için.
  • Oyun ilerlemesi: Oyun skorları, tamamlanan düzeyler, doğru/yanlış cevap sayıları.
  • Tercihler: Ses, büyük yazı tipi, anksiyete dostu mod, ruh hali.
  • Cihaz tanımlayıcı: Rastgele üretilen (kişisel olmayan) UUID. Yalnız premium kod doğrulamasında sunucuya gönderilir.
  • Hata telemetrisi (opsiyonel): Yalnız uygulama çöktüğünde hata mesajı + yığın izi. IP adresi ve kişisel veri içermez.

Toplamadığımız: Konum, kontaklar, fotoğraflar, mikrofon kaydı, reklam kimliği, kamera.

3. Verileriniz Nerede Saklanır?

Tüm kişisel veriler yalnızca kullandığınız cihazın yerel depolama alanında (localStorage / IndexedDB) saklanır. Sunucuya gönderilmez.

İstisna: Premium kod doğrulaması sırasında kod ve cihaz UUID'niz (ad/yaş DEĞİL) sunucuya HTTPS ile gönderilir. Sunucu yalnızca kodun geçerliliğini kontrol eder ve cihaz sayısını takip eder.

Veri silme: Tarayıcı verilerinizi veya uygulama verisini silerseniz tüm kayıtlar kalıcı olarak kaldırılır. Uygulama içindeki Ayarlar → Verimi Sil butonu aynı işlemi yapar.

4. Çocuk Kullanıcıların Korunması

Uygulama 6–12 yaş çocuklar için tasarlanmıştır. COPPA ve KVKK gereklerine uygun olarak:

  • İlk açılışta veli / yasal temsilci onayı alınır.
  • Hiçbir reklam, 3. taraf izleme veya sosyal medya entegrasyonu YOKTUR.
  • Çocuğa ait veriler pazarlama amacıyla KULLANILMAZ, SATILMAZ.
  • Veli istediği an tüm verileri silebilir (Ayarlar → Verimi Sil).

5. Haklarınız (KVKK md. 11)

Kanunen aşağıdaki haklara sahipsiniz:

  • Verilerinizin işlenip işlenmediğini öğrenme
  • Verilerinize erişme ve kopyasını talep etme
  • Eksik/yanlış verilerin düzeltilmesini isteme
  • Verilerin silinmesini (unutulma hakkı) talep etme
  • Otomatik işleme sonucunda oluşan analizlere itiraz etme

Bu hakları kullanmak için destek@matbil.app adresine başvurabilirsiniz. En geç 30 gün içinde yanıt verilir.

6. Veri Güvenliği

  • Sunucu iletişimi her zaman HTTPS/TLS üzerinden yapılır.
  • Premium durumu imzalanmış (tamper-proof) şekilde saklanır.
  • Yönetici erişimi SHA-256 + salt hash ile korunur, brute-force koruması vardır.
  • Uygulama güncellemelerinde veri şeması migration ile güvenli dönüştürülür.

7. Değişiklikler

🛡️ MatBil Privacy Policy

Version 1.0 • Effective: 2026-04-24 • GDPR & COPPA compliant

Summary: MatBil is an educational app for mathematical cognition and dyscalculia. All data we collect is stored only on your device. Nothing personal is sent to our server (except premium code validation). We never share data with third parties.

1. Data Controller

MatBil is developed by Prof. Dr. Yılmaz Mutlu & Prof. Dr. Sinan Olkun for academic/educational purposes.

Contact: destek@matbil.app

2. Data We Collect

  • Student profile: Name (or nickname), age — to personalize the game experience.
  • Progress: Game scores, completed levels, correct/incorrect answers.
  • Preferences: Sound, large text, anxiety-friendly mode, mood.
  • Device UUID: Randomly generated (non-personal). Sent to server only for premium code validation.
  • Error telemetry (optional): Only on crashes. No IP, no personal data.

We do NOT collect: location, contacts, photos, microphone, advertising ID, camera.

3. Where Is Data Stored?

All personal data is stored only in your device's local storage (localStorage / IndexedDB). Nothing is sent to our server.

Exception: When validating a premium code, the code and your device UUID (NOT name/age) are sent over HTTPS for verification and device-count limiting.

Data deletion: Clearing browser data or using Settings → Delete My Data removes everything permanently.

4. Children's Privacy (COPPA / GDPR-K)

Designed for ages 6–12. Compliance measures:

  • Parental consent required on first launch.
  • NO ads, NO third-party tracking, NO social media integrations.
  • Child's data is never used for marketing, never sold.
  • Parents can delete all data anytime (Settings → Delete My Data).

5. Your Rights

  • Know whether your data is being processed
  • Access and request a copy of your data
  • Request correction of inaccurate data
  • Request deletion (right to be forgotten)
  • Object to automated processing

To exercise these rights: destek@matbil.app (reply within 30 days).

6. Security

  • All server communication is HTTPS/TLS.
  • Premium state is cryptographically signed (tamper-proof).
  • Admin access protected by SHA-256 + salt hash, with brute-force protection.
  • Data schema migrations preserve user data across app updates.

7. Changes

This policy may be updated. Significant changes trigger an in-app notification and re-consent.
