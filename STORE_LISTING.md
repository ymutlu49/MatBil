## 🇬🇧 English Text

### Short Description (80 chars)
```
From number sense to spatial thinking: science-based math games for dyscalculia.
```

### Full Description (4000 chars)
```
🧠 MatBil — Math Cognition Foundations

An educational app designed for children aged 6-12, featuring 35+ science-based games for dyscalculia (math learning difficulty) and math anxiety.

📚 WHO IS IT FOR?
• Elementary school students (grades 1-4)
• Children with math learning difficulties
• Anyone wanting to build number sense, estimation, and geometric cognition
• Teachers: class management and reporting
• Parents: track child's development

🎮 WHAT'S INSIDE?
• Subitizing: Dot Hunter, Memory, Grouping Master, Finger Bridge, Lucky Finger
• Arithmetic: Addition/Subtraction Strategies, Number Bonds, Part-Whole, Number Walk
• Symbolic Processing: Place Value, Multiple Representations, Code Translator, Bigger-Smaller
• Geometry (Van Hiele): Shape Recognition, Symmetry Mirror, Mental Rotation, Direction & Distance
• Estimation: Estimation Jars, Number Line, Measurement Estimate, Computational Estimate
• Cognitive Skills: Numerical Memory, Numerical Stroop, Task Switching

🏅 SCIENTIFIC FOUNDATION
• Based on research by Prof. Dr. Yılmaz Mutlu & Prof. Dr. Sinan Olkun
• Classic pedagogical models: Shepard-Metzler, Van Hiele, Kaufman, Clements
• Each game is mapped to book chapters, theory to practice

✨ FEATURES
• 🐢 Calm Mode: anxiety-reducing versions without time pressure
• A+ Large Text: accessibility for dyslexia/vision
• 🔊 Turkish audio guide
• 📊 Detailed development reports (parent & teacher)
• 🏆 Badges and levels (motivation)
• 🎒 Classroom management (for teachers)
• 👨‍👩‍👧 Multiple student profiles

🛡️ PRIVACY
• All data stored on your device only
• Nothing sent to server (except premium code)
• No ads, no tracking, no third-party sharing
• GDPR / COPPA compliant, parental consent required
• "Delete My Data" button anytime

🎓 PREMIUM
Book code unlocks all 35 games. Free version includes 1 representative game per category. Codes are for owners of "Math Cognition Foundations" book only.

💬 SUPPORT
destek@matbil.app
```
